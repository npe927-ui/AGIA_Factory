# 03 CANALES

Especialización por canal para evitar copy genérico.

## CANALES INCLUIDOS:
- **cold_email**: Prospección y apertura.
- **emkd**: Email marketing diario y relacional.
- **landing_pages**: Conversión directa.
- **web_texts**: Home, About us, Service pages.
- **sales_letters**: Cartas de venta de largo formato.
- **budgets**: Presupuestos que cierran.
- **business_bios**: Perfiles profesionales persuasivos.
- **reports**: Informes técnicos con tono de autoridad.
