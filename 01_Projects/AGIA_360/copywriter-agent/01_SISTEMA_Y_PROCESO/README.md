# 01 SISTEMA Y PROCESO

Este directorio contiene los flujos de trabajo y marcos técnicos del Agente IA Copywriter.

## ARCHIVOS:
- **Flujo de trabajo**: Pasos secuenciales desde la recepción del brief hasta la entrega final.
- **Checklist antes de escribir**: Verificaciones de audiencia, ángulo y oferta.
- **Checklist de revisión**: Control de calidad, ritmo, claridad y CTA.
- **Frameworks de trabajo**: Implementaciones de AIDCA, PAS, Storytelling, Manejo de Objeciones y Estructura de Oferta.
