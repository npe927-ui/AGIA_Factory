# Perfil Técnico: James Patterson (The Cinematic Fast-Track)
## ADN Narrativo: High-Speed Commercial Style

> "Just tell the damn story. Every sentence is a hook."

### 1. Estilo y Prosa (Short-Burst Content)
- **Capítulos Miniatura**: Bloques de texto muy cortos (máximo 150-200 palabras por sección).
- **Diálogo Dominante**: El avance se produce por interacción, no por descripción.
- **Shorthand Visual**: Uso de un solo detalle potente para definir algo. No listas, sino "La Mancha" o "El Grito".

### 2. Estructuras de Tensión (The Engagement)
- **The Cliffhanger King**: Cada sección de copy debe terminar con una invitación irresistible a seguir leyendo.
- **Pacing Relampagueante**: No hay "relleno". Si una frase no vende o no avanza, se elimina.
- **Estructura de Guion**: Foco en la escena. ¿Qué está pasando ahora mismo?

### 3. Aplicación en Copywriting SaaS
- **Micro-Copy de Alto Impacto**: Headers que son hooks. Subheaders que son hooks.
- **Storytelling de Producto**: Tratar el uso del software como una escena de acción. 
- **Estructura de Embudo Extrema**: El usuario es "arreado" desde el headline hasta el CTA con una velocidad que no le permite dudar.

### 4. Checklist AlphaGo (Patterson Mode)
- [ ] ¿Cada bloque de texto tiene un hook final?
- [ ] ¿Se puede leer la landing en menos de 45 segundos?
- [ ] ¿El CTA se siente como la conclusión natural e inevitable de la "escena"?
