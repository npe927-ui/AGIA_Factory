# Dataset de Copywriting Troncal

Este dataset contiene la literatura esencial para el entrenamiento y operación del agente copywriter.

## Estructura

Todos los archivos han sido normalizados y certificados bajo estándares profesionales de Markdown (.md), cumpliendo con:

- **MD001**: Jerarquía de encabezados lógica y secuencial.
- **MD022**: Espaciado correcto alrededor de encabezados.
- **MD041**: Inicio de archivo con encabezado de primer nivel (H1).
- **MD047**: Terminación de archivo con un único salto de línea.

### Contenido

- Libros de autores clásicos (Ogilvy, Hopkins, Schwartz).
- Libros de autores modernos (Isra Bravo, Rosa Morel).
- Fichas de metodología (Sugarman, Schwartz).
