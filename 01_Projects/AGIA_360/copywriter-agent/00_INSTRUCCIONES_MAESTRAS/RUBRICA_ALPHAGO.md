# Rúbrica AlphaGo: Auditoría de Copy

Esta es la rúbrica oficial para la Fase 2 (Evaluación) y Fase 3 (Self-play) del Agente Copywriter. Cada pieza de copy generada debe ser sometida a este escrutinio técnico.

| Criterio | Peso | Descripción Técnica | Puntuación (1-5) |
| :--- | :---: | :--- | :---: |
| **Open Loop** | 20% | ¿Plantea una pregunta o conflicto que obliga a seguir leyendo? (Técnica Dan Brown/Patterson) | |
| **Ritmo** | 20% | ¿Existe variación staccato? ¿Alternancia entre frases cortas de impacto y largas de inmersión? (Técnica Child/Hemingway) | |
| **Move 37** | 20% | ¿El ángulo es inesperado? ¿Evita el cliché de IA y aporta una perspectiva única del problema? | |
| **CTA Emocional** | 20% | ¿La venta fluye naturalmente desde la narrativa o se siente como un "pegote" comercial? | |
| **Voz (Motor)** | 20% | ¿Es coherente con el autor elegido? ¿Aplica las leyes del manifest específico? | |

## Escala de Evaluación:
- **9.0 - 10.0 (Elite)**: Listo para envío masivo. "Movimiento 37" detectado.
- **7.0 - 8.9 (Aceptable)**: Requiere un pulido menor en ritmo o CTA.
- **< 7.0 (Fallido)**: El agente debe re-generar el output usando un motor literario diferente.

## Instrucción para el Orquestador:
El agente evaluador debe citar específicamente qué parte de la Rúbrica ha fallado o destacado, utilizando fragmentos del texto como prueba.
