# diferencias entre páginas de venta y Docs

Anoche, un lector y cliente llamado Antonio me preguntó algo muy acertado: «¿Qué diferencia este Doc de una página de ventas, teniendo en cuenta que ambas pueden tener la misma longitud de texto, sean mil, 2mil o 20mil palabras?» Es una buena pregunta. Porque no hay una, ni dos ni tres, sino tres diferencias importantes. (Si no viste Malviviendo, no eres digno de entender este chiste). Te cuento a nivel práctico cuáles son: #1 Velocidad Puedes crear un Doc en 30 minutos, una página requiere más tiempo y cierto conocimiento técnico y de diseño (aparte de tener una web, contratar un hosting o una plataforma que las haga) lo cual para validar ideas nuevas supone una diferencia bestial. Sí, ahora existe la posibilidad de hacerlo con la IA. Aun así, eso requiere 1) saber algo de código y cómo montarlo luego en HTML, 2) pagar por las herramientas, y 3) tener ciertas nociones de cómo debe ser a nivel visual una BUENA página de ventas. Y cuando necesitas cambiar algo sobre la oferta... Puedes editar un Doc en dos minutos en función de los comentarios de los clientes potenciales sin tocar código ni un maquetador de páginas. #2 Diseño Las páginas de venta se tienen que maquetar de una cierta manera para que se vean bien. Buscan verse profesionales, bonitas, llamar la atención. El Doc busca la simplicidad y la claridad a la hora de cerrar la venta. Tiene CERO diseño, a propósito. Proyecta el estatus de la oferta y está pensado para Patriotas que ya saben lo que quieren comprar, solo quieren que les hagan la oferta adecuada. #3 Estructura Una página de ventas usa una estructura más desarrollada, pensada para calentar y convencer al tráfico frío/templado. Profundiza en el problema, lo agita, conciencia al lector sobre la solución, tiene una amplia sección de bullet points, etc. Eso está bien cuando lo que quieres es que la página de ventas haga todo el trabajo pesado de venta y generar confianza desde cero para comprar. El Doc está diseñado para terminar una conversación que ya estás teniendo. No pretende convencer, muestra dónde está la puerta y explica lo que hay detrás de ella. En resumen: Si vas a vender productos low ticket a gente que no te conoce, usa una página de ventas. Si vas a vender productos más caros a gente que ya te ha comprado, usa un High Ticket Doc. De todas formas, si un Doc funciona muy bien, siempre puedes convertirlo en una página de ventas más adelante.
