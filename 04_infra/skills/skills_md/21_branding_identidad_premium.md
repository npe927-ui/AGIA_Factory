# Skill: Branding e Identidad Premium — Siglo XXI (AI-Enhanced)

## Propósito
Establecer un estándar de diseño para la AGIA Factory que evite los clichés del siglo XX y proyecte autoridad, tecnología y precisión quirúrgica. Esta skill guía la generación de activos visuales mediante IA para que parezcan diseñados por una agencia boutique de 2026.

## Principios de la "Identidad Esmeralda"

### 1. El Princio del "Easter Egg" (Significado Oculto)
- No uses iconos literales (no dibujes un camión para una logística).
- Usa formas negativas o ligaduras tipográficas que sugieran el concepto (ej: la flecha oculta de FedEx).
- El diseño debe premiar al observador atento.

### 2. Minimalismo Industrial vs. "Stock Design"
- **NO**: Sombras suaves, degradados de arcoíris, iconos redondeados "friendly".
- **SÍ**: Materiales táctiles (acero cepillado, fibra de carbono, cristal esmeralda), bordes afilados, precisión láser.
- La estética debe ser "Aegis": protectora, robusta y tecnológica.

### 3. Paleta de Color "Sub-Zero"
- **Base**: Obsidiana Mate (#1A1A1B) o Gris Ártico (#E5E7EB).
- **Accento Primario**: Verde Esmeralda (#059669) — representa eficiencia y vida tecnológica.
- **Accento Táctico**: Rojo Láser (#E31E24) — solo para puntos de precisión o alertas críticas.
- **Evita**: La mezcla 50/50 de verde y rojo (parece Navidad o vegetales). El rojo debe ser < 5% del logo.

### 4. Tipografía Bespoke (Custom)
- Usa fuentes con mucho *kerning* (espaciado).
- Las letras deben parecer esculpidas, no escritas.
- Mezcla pesos (Bold para la potencia, Light para la tecnología).

## Protocolo Anti-Cliché (Filtro de Calidad)

| Si el negocio es... | EVITA estos iconos | USA este concepto |
|---|---|---|
| Logística | Camiones, globos, cajas | Flujo cinético, escudos aerodinámicos |
| Frigorífico | Copos de nieve, pingüinos | Cristales de energía, prismas de hielo |
| Seguridad | Candados, llaves | Geometría de interceptación, nodos de red |
| Estrategia | Piezas de ajedrez, cerebros | Monolitos, flechas vectoriales de impacto |

## Prompt Engineering para Logos V8+

Para obtener resultados premium, el prompt debe incluir:
1. **Materialidad**: "Brushed titanium", "Emerald frosted glass".
2. **Iluminación**: "Hard rim lighting", "Cinematic obsidian backdrop".
3. **Estilo**: "Swiss School of Design meets Cyber-Industrialism".
4. **Instrucción Negativa**: "No photorealistic trucks, no cartoonish icons, no rounded corners".
