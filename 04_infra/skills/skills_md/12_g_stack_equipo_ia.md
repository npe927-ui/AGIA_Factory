# Skill: G Stack — Tu Equipo de IA (25 Habilidades de Gestión)

## Propósito
Sistema de 25 habilidades especializadas para gestionar proyectos completos con Claude. Orquesta el trabajo de Pau, Ethan y Alma desde una sola interfaz de mando.

## Las 25 Habilidades organizadas en 5 áreas

### Área 1 — Planificación Estratégica (5 habilidades)
1. **Visión de proyecto** → Define el norte del proyecto en 1 párrafo
2. **OKRs** → Genera Objetivos y Resultados Clave medibles
3. **Roadmap** → Plan de fases con milestones y dependencias
4. **Risk analysis** → Identifica los 5 mayores riesgos y mitigaciones
5. **Decision log** → Registra decisiones arquitectónicas con contexto

### Área 2 — Gestión de Tareas (5 habilidades)
6. **Sprint planning** → Divide el trabajo en sprints de 1 semana
7. **Task breakdown** → Convierte epics en tareas accionables
8. **Blocker detection** → Identifica qué está bloqueando el avance
9. **Progress report** → Genera reporte de avance para stakeholders
10. **Retrospectiva** → Analiza qué funcionó y qué mejorar

### Área 3 — Comunicación (5 habilidades)
11. **Briefing de cliente** → Documento de onboarding para nuevos clientes
12. **Status update** → Email de actualización semanal
13. **Propuesta comercial** → Documento de propuesta con precios y alcance
14. **Meeting agenda** → Agenda estructurada para reuniones eficientes
15. **Handoff document** → Traspaso de información entre agentes/personas

### Área 4 — Análisis y Datos (5 habilidades)
16. **Competitive analysis** → Análisis de competidores con Tavily
17. **Market research** → Investigación de mercado automatizada
18. **KPI dashboard** → Define métricas clave por tipo de proyecto
19. **User feedback analysis** → Extrae insights de feedback de usuarios
20. **ROI calculator** → Calcula retorno de inversión de acciones

### Área 5 — Operaciones Factory (5 habilidades)
21. **BUNKER sync** → Actualiza el BUNKER con el estado actual
22. **Agent orchestration** → Asigna tareas a Pau, Ethan o Alma
23. **Deploy checklist** → Verifica que todo está listo antes del deploy
24. **Security audit** → Revisa vulnerabilidades según NORMATIVA_Y_SEGURIDAD.md
25. **Factory snapshot** → Genera foto del estado completo de la Factory

## Cómo activar cualquier habilidad

```
G STACK — HABILIDAD [NÚMERO]: [NOMBRE]
PROYECTO: [Nombre del proyecto]
CONTEXTO: [Información relevante]
OUTPUT ESPERADO: [Qué necesitas]
```

## Ejemplo — Habilidad 21: BUNKER sync

```
G STACK — HABILIDAD 21: BUNKER sync
PROYECTO: AGIA 360 / SaaS Factory
CONTEXTO: Hoy completamos el Bloque 2 de skills y arreglamos la memoria de Supabase
OUTPUT ESPERADO: Nueva entrada en el LOG del BUNKER con formato estándar
```
